# Portfolio

Personal developer portfolio showcasing selected projects, technical skills, and contact information.

## Highlights
- Focus on full-stack application development
- Projects built with React, TypeScript, Flask, and API integrations
- Responsive UI layouts and clean component structure

## Project Structure
- `index.html` - Main page content and structure
- `styles.css` - Visual styling and responsive layout
- `Images/` - Portfolio project assets

## Run Locally
1. Clone this repository.
2. Open `index.html` in your browser.

Optional:
- Run a local static server for easier iteration:
  `npx serve .`
